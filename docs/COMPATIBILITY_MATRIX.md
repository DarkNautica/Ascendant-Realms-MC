# Compatibility Matrix

Audit date: 2026-06-13

Sources used for this first pass: Modrinth project/version metadata where available, with unresolved or likely CurseForge/resource-pack-only items marked for manual verification. This is an audit, not an install manifest.

Status meanings:

- Strong Yes: good fit for the preliminary 1.20.1 Forge path or foundational enough to test early.
- Maybe: compatible signal exists, but needs careful batch testing or exact provider verification.
- Delay: compatible or promising, but too risky/heavy for early batches.
- Reject: wrong loader/version, archived/deprecated, duplicate, or not aligned enough for the main path.
- Installed: active in Packwiz for Batch A, Batch B, Batch C, Batch D, Batch E1, Batch E2, Batch F, Batch G, Batch H, Batch J, Batch K, Batch L, Batch M, or Batch N.
- Config tree: active pack source under `config/puffish_skills/`.
- Planned: approved for planning only; not installed until Jayden approves the batch install.
