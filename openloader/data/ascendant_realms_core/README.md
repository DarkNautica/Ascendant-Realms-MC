# Ascendant Realms Core Runtime Helpers

This Open Loader datapack exposes small `/function ascendant_core:*` helpers for testing the live core bridge.

It does not alter world generation, loot, mobs, recipes, or structure pools.

Useful test commands:

- `/function ascendant_core:status`
- `/function ascendant_core:debug/add_structure_clear`
- `/function ascendant_core:debug/add_boss_proof`
- `/function ascendant_core:debug/add_dragon_proof`
- `/function ascendant_core:debug/reset_self`
