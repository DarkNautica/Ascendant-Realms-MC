# Ascendant Realms Skills Datapack

This datapack is the fallback/source copy of the custom Ascendant Realms progression tree for Puffish Skills.

The primary pack load path is now:

```text
config/puffish_skills/
```

Load path for a server or local world:

```text
<world>/datapacks/ascendant_realms_skills/
```

The active tree lives under `data/ascendant_realms/puffish_skills/` and uses only existing item icons plus confirmed attribute rewards. No custom resource pack is required for this pass.

Default Skill Trees was removed from the active Packwiz mod list so the custom tree becomes the intended progression surface. Use the datapack copy only if the config auto-load path fails.
