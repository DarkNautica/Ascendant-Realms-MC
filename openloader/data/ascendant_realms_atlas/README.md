# Ascendant Realms Atlas Datapack

Generated by `scripts/generate-ascendant-atlas.py`.

This pack keeps the first Atlas-authored region waymarks as debug/source assets and adds `/function ascendant_atlas:status`. Natural waymark placement is disabled.

As of 2026-06-18, the Atlas Overworld dimension override and Atlas continentalness/land-bias override are intentionally removed. Random Tectonic/Terralith generation is the active terrain baseline. The `ascendant_atlas_regions` helper may remain installed for diagnostics/history, but this datapack must not reference `ascendant_atlas_regions:regional_multi_noise` or `ascendant_atlas_regions:atlas_land_bias` while `config/ascendant_atlas/worldgen_override_policy.json` has `worldgen_override_enabled=false`.
