# Ascendant Guild Data

This folder contains safe data/design scaffolds for the Guild/Hunter system. These JSON files are not a live quest database yet. They define the rank ladder, rival hunters, bounty categories, NPC roles, and Hunter Board blueprints so FTB Quests, FTB Ranks, Easy NPC, CustomNPCs, Bountiful, and KubeJS can be wired deliberately after boot validation.
