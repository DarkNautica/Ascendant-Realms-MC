# Datapacks Folder

Fallback/source datapacks:

- `ascendant_realms_skills/` defines the unified Ascendant Web Puffish Skills progression tree.
- `ascendant_realms_world_integration/` mirrors the active OpenLoader world-integration crash-fix datapack.

Primary auto-load path:

- `config/puffish_skills/`

Future datapacks may also be used for:

- loot table tuning
- structure spacing
- spawn rules
- tags
- progression gates

Server/local world fallback path:

```text
<world>/datapacks/ascendant_realms_skills/
```

Active OpenLoader path for global world-integration fixes:

```text
config/openloader/data/ascendant_realms_world_integration/
```

Do not add unrelated datapacks until the current skill tree plus Batch K/L/M/N validation gates pass.
